<h1 class="h3" id="the-worlds-largest-open-source-foundation">The World's Largest Open Source Foundation<a class="headerlink" href="#the-worlds-largest-open-source-foundation" title="Permalink">&para;</a></h1>
</div>
<div class="col-sm-4 col-sm-offset-2">
<ul>
<li>All volunteer community</li>
<li>227M+ lines of code in&nbsp;stewardship</li>
<li>4.2B+ lines of code&nbsp;changed</li>
<li>4.1M+ code commits</li>
<li>820 individual ASF&nbsp;Members</li>
<li>8,100+ Apache Committers</li>
<li>40,000+ code contributors</li>
<li>488,000+ people involved in our&nbsp;communities</li>

</div>

<ul>

<li>35M+ weekly page views across <a href="/">apache.org</a></li>
</ul>

<section class="bg-info">
<div class="container">
<p>The ASF develops, shepherds, and incubates hundreds of freely-available, enterprise-grade projects that serve as the
backbone for some of the most visible and widely used applications in computing today. Through the ASF's merit-based
process known as <a href="/theapacheway">"The Apache Way,"</a> more than <a href="http://community.zones.apache.org/map.html">800 individual volunteer Members and 8,100+ code
Committers</a> across six continents successfully collaborate on innovations
in Artificial Intelligence and Deep Learning, Big Data, Build Management, Cloud Computing, Content Management, DevOps,
IoT and Edge Computing, Mobile, Servers, and Web Frameworks, <a href="https://projects.apache.org/projects.html?category">among other categories</a>.</p>
<p><strong>APACHE IS &hellip;</strong></p>
<p><strong>OPEN:</strong> The Apache Software Foundation provides support for 350+ Apache Projects and their Communities,
furthering its mission of providing Open Source software for the public&nbsp;good.</p>
<p><strong>INNOVATION:</strong> Apache Projects are defined by <a href="/theapacheway">collaborative, consensus-based processes</a>, an open,
pragmatic software license and a desire to create high quality software that leads the way in its&nbsp;field.</p>
<p><strong>COMMUNITY:</strong> We are a <a href="http://community.apache.org/">community of developers and users</a> of enterprise-grade,
Open Source Apache projects used in every Internet-connected country on the&nbsp;planet.</p>
</div>
